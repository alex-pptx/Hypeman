# Hypeman

Hypeman is a client addon for Archeage Classic. The goal of Hypeman is provide a kill counter and combo-meter like you'd find in a game like ultrakill, devil may cry, etc. Grinding mobs for coinpurses makes my brain go numb so I wanted to make an addon to hype me up and give me a constant feed of dopamine to keep going!
